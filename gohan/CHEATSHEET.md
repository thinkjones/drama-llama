# 🐍 Activity 2: Welcome to Python — Cheatsheet

**Your mission:** Use **Claude Code** (an AI coding buddy) to write your very
first Python program, then make it your own!

---

## 1. Open the terminal and go to this activity's folder

Open **Ubuntu** from the Start menu (same as Activity 1), then type:

```bash
cd ~/drama-llama/gohan
```

That moves you into the `gohan` folder, where your program will live.

---

## 2. Start Claude Code

Claude Code is an AI that writes code with you, right in the terminal. Start it
by typing:

```bash
claude
```

Wait a moment for it to load. When you see a prompt where you can type, you're
ready to talk to Claude! 🤖

> First time? It may ask you to log in — your instructor will help.

---

## 3. Ask Claude to build your program

Type this exactly and press **Enter**:

> Write a simple Python program called `agent_welcome.py` that asks for the
> user's agent name and prints a cool welcome message with ASCII art.

Claude will create a file called `agent_welcome.py`. If it asks for permission
to create the file, say **yes**.

---

## 4. Run your program

Once the file is made, run it by typing:

```bash
python3 agent_welcome.py
```

Type your secret **agent name** when it asks, and watch the cool welcome
message appear! 🎉

---

## 5. View and edit the file in Zed

**Zed** is a friendly text editor. To open your program in it, type:

```bash
zed agent_welcome.py
```

This opens the file in a window where you can read and change the code.

---

## 6. Make it YOURS 🎨

Ask Claude Code to customize the ASCII art. Try prompts like:

- "Change the ASCII art to a llama."
- "Make the welcome message rainbow colored."
- "Add my favorite emoji to the message."
- "Make it print three different jokes."

After each change, run it again with `python3 agent_welcome.py` to see what
happened. Experiment and have fun!

---

### 💡 Remember
- `cd` = change folder · `ls` = list files · `claude` = start Claude Code
- `python3 filename.py` = run a Python program
- `zed filename.py` = open a file in the Zed editor
